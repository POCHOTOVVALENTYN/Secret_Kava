# План реалізації: Покращення розділу відгуків та NPS

Ми реалізуємо комплексне покращення роботи з відгуками клієнтів:
1. Збір оцінки 1-5 зірочок через FSM-сценарій при написанні відгуку.
2. Синхронізація відгуків з Google Sheets (аркуш `"Реєстр відгуків"`).
3. Розділ модерації відгуків у меню адміністратора.
4. Додатковий запит текстового фідбеку після проходження NPS-опитування.

## Proposed Changes

### Бот та обробка відгуків

#### [MODIFY] [reviews.py](file:///Users/valentin/Secret_Cava/app/bot/handlers/reviews.py)
- **Зміни**:
  - Розширити `ReviewFSM` станами `SelectRating` та `EnterReviewText`.
  - У `reviews:write` надсилати inline-клавіатуру вибору зірочок (1-5).
  - При виборі оцінки зберігати її в FSM та запитувати текст відгуку.
  - При збереженні відгуку (`save_client_review`) записувати його в БД та додавати новий рядок на аркуш `"Реєстр відгуків"` зі статусом `"На модерації"`.
  - Оновити `process_nps_rating`: якщо оцінка низька (<= 6), пропонувати написати зауваження. Якщо висока (>= 9), пропонувати написати відгук для публікації.

#### [MODIFY] [admin.py](file:///Users/valentin/Secret_Cava/app/bot/handlers/admin.py)
- **Зміни**:
  - Додати кнопку `"📝 Модерація відгуків"` до головного меню адмінки (`_kb_main()`).
  - Додати callback-хендлер `adm:reviews` для перегляду немодерованих відгуків по одному.
  - Додати хендлери для схвалення (`adm:rev:approve:{id}`) та відхилення (`adm:rev:reject:{id}`) відгуків.
  - При схваленні оновлювати статус у БД на `is_moderated=True`, оновлювати статус в Google Sheets на `"Опубліковано"`, та надсилати сповіщення користувачу.
  - При відхиленні видаляти відгук з БД та оновлювати статус в Sheets на `"Відхилено"`.

## Verification Plan

### Manual Verification
- Створити відгук через бота, вибравши оцінку 4 зірочки. Переконатися, що рядок з'явився у гугл-таблиці `"Реєстр відгуків"` в статусі `"На модерації"`.
- Зайти в адмін-меню під ADMIN_ID, перейти в розділ модерації, схвалити відгук. Перевірити, що статус у Sheets змінився на `"Опубліковано"`, а відгук з'явився на головній сторінці відгуків.
- Перевірити отримання сповіщення користувачем.
