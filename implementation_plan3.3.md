# План реалізації: Рефакторинг розділу «Оренда всього простору»

Ми впровадимо повноцінний лідогенератор для оренди всього терапевтичного простору:
1. Додамо FSM стани для збору імені та номера телефону.
2. Реалізуємо збереження заявки в Google Sheets на новий аркуш `"Заявки на оренду простору"`.
3. Реалізуємо надсилання миттєвих сповіщень адміністраторам.
4. Забезпечимо розумне очищення чату (редагування головного повідомлення та видалення вводу користувача).
